## Install

- Node.js 12+ is required.

### NPM

To install via NPM:

```
npm install @digitalbazaar/core-document-loader
```

### Development

To install locally (for development):

```
git clone https://github.com/digitalbazaar/core-document-loader.git
cd core-document-loader
npm install
```
